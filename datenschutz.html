Wie werden Ihre Daten erhoben?

Ihre Daten werden zum einen dadurch erhoben, dass Sie uns diese mitteilen. Hierbei kann es sich z. B. um Daten handeln, die Sie uns per E-Mail schicken. Andere Daten werden automatisch beim Besuch der Website durch unsere IT-Systeme bzw. durch die Systeme unseres Providers erfasst. Das sind vor allem technische Daten (z. B. Internetbrowser, Betriebssystem oder Uhrzeit des Seitenaufrufs). Die Erfassung dieser Daten erfolgt automatisch, sobald Sie unsere Website aufrufen.

Wofür werden die Daten genutzt?

Ein Teil der Daten wird erhoben, um eine fehlerfreie und komfortable Bereitstellung der Website zu gewährleisten. Andere Daten können zur anonymisierten Analyse Ihres Nutzerverhaltens verwendet werden.
